How to Use with Exeunt

Testing against this page

  1. Serve this directory with python3 -m http.server 8000
  2. Open http://localhost:8000/demo/ in Chrome
  3. Install and enable the Exeunt extension
  4. Click the Exeunt icon in the toolbar
  5. Exeunt should detect SCORM 1.2 or 2004 depending on the mode
  6. Use "Complete" in Exeunt, then watch the log and CMI state update

Switching modes

Use the toggle above to switch between SCORM 1.2 and 2004. Switching resets all state and swaps which API object lives on window, so Exeunt detects the correct standard.

Why not file://? Chrome does not inject content scripts into file:// URLs by default. You must serve this page over HTTP (even localhost) for Exeunt to interact with it.
