The argument

Compliance recording and learning are not the same thing.

An LMS is an excellent audit trail. It was designed to record completion, not to produce capability. The SCORM standard was written in 2001, when the primary question was "did the learner finish?" not "can the learner perform?"

These are different questions. They require different answers. We have spent two decades and hundreds of billions of dollars pretending they are the same question.

Exeunt is for the developers testing course integrations, the designers iterating on builds, and the learners who demonstrably hold the knowledge and see no reason to prove it by clicking slides.

The name

Exeunt. They all exit.

/ˈɛk.sɪ.ʌnt/ - Latin, third-person plural

Exeunt is a stage direction. It appears in the scripts of plays - Shakespeare's especially - to signal that all remaining characters leave the stage. Not one actor. Everyone. The scene is over.

It felt like the right word for a tool that ends the performance.

All the world's a stage,
And all the men and women merely players;
They have their exits and their entrances.
- As You Like It, Act II, Scene VII
Capabilities

Robust by design.

⟳
Auto-retry loop
Scans up to N times with configurable delay. Handles the 1–3 second window most courses take to initialise their SCORM API.
◎
Multi-location search
Checks window.API, window.top.API, window.parent.API, pipwerks shim, and SCORM 2004 equivalents across all accessible frames.
◈
SCORM 1.2 + 2004
Both standards, all four SCORM 2004 editions, correct write order, passed/completed fallback, full error code annotation.
✓
Pre-close verification
Reads values back after writing, before Terminate, to confirm they actually stuck. Every return value logged with human-readable error strings.
⊡
Configurable
Score, session time, status preference, retry behaviour, suspend data clearing, auto-complete mode. Everything you need for thorough testing.
◷
Zero dependencies
Vanilla JS. No build step. No frameworks. No remote code. No analytics. No tracking. Nothing phoning home, ever.
Install

Up in under a minute.

1
Download the latest release and unzip it.
2
Navigate to chrome://extensions and enable Developer mode.
3
Click Load unpacked and select the extension/ folder.
4
Open any SCORM course, click the Exeunt icon, click Complete Course.
